Standard License

Here's the gist:

With a Standard license, you can use this module on one instance of DNN.

It can be used on all of the portals for that instance, but has to stay with that one instance. If you need to move it to another instance of DotNetNuke, you will need to uninstall it from the other instance. This only applies to productions sites, you can use the module on staging and testing sites, in addition to your one production site that the license allows.

For the full details, please see the full license (included with this module package, or you can view it on our website).
